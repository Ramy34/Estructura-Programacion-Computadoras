# Estructura y Programación de Computadoras

Repositorio de la clase **Estructura y Programación de Computadoras**. Este espacio contiene los ejercicios, prácticas de laboratorio, proyectos y apuntes desarrollados a lo largo del curso.

## 📖 Descripción del Curso

En este repositorio se aborda el estudio del hardware y el software desde un bajo nivel. Incluye temas como la arquitectura de computadoras (modelo de Von Neumann), representación de datos, organización de la memoria y la relación directa entre el lenguaje ensamblador y lenguajes de nivel medio.

## 🛠️ Tecnologías y Herramientas Utilizadas

*(Ajusta estas herramientas según lo que utilices)*
* **Lenguajes:** C, C++, Lenguaje Ensamblador (x86, ARM, RISC-V o MIPS).
* **Simuladores:** Logisim, MARS, SPIM, o Proteus.
* **Compiladores:** GNU GCC, NASM.
